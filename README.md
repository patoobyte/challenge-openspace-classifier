## ✨ Description ✨

A simple script that takes a list of names and randomly assigns seats in an openspace setting.

```
├── utils/
│   ├── file_utils.py
│   ├── openspace.py
│   └── table.py
├── main.py
├── new_colleagues.txt
├── notebook_guide.ipynb
└── README.md
```

## 💻 Installation & Usage 🖱️

1. Clone the repository to your local machine.

2. Run the script with  
```
python main.py
```
or 
```
python3 main.py
```

3. The script:  
	- Reads the input file (*new_colleagues.txt*)
	- Randomly assigns seats to everyone  
		- If there are not enough seats, the script will end.
	- Displays the seat arrangement on the console
	- Stores the seat arrangement in a .csv file.
